# 我们都是AI
这是一款大型模拟经营类网络游戏
主要模拟真实世界的玩法，使用电脑硬件参与游戏，没有硬件玩不了（只能查看游戏世界）。
硬件可以是esp8266、esp32、PC、手机端
游戏内有大量职业可以创建，每种职业都可以做相关的工作，通过设备供电进行工作，断电即为离线状态，游戏世界通过后台逻辑自动运行，高仿真模拟整个世界，不断修复逻辑，给玩家更好的体验，有兴趣的可以下载硬件端参与游戏。

后续更新一直在进行，这是一款长期维护的大型多人游戏。
一起来打造新的世界吧！
